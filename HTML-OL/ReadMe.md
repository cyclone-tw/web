🧑‍🏫 代課教師交辦事項生成器

這是一個單一的 HTML 檔案網頁應用程式，旨在幫助教師在出差時代課老師準備清晰、系統化的每日交辦事項清單。它採用暗色系設計，介面簡潔，專為快速生成和列印文件而設計。

✨ 主要功能

日期選擇： 選擇出差日期，自動識別星期幾。

課表自動導入： 根據內建的 weeklySchedule 數據，自動帶入當日的課表（時段、科目、班級）。

交辦事項輸入： 提供可編輯的區域，讓您可以為每節課輸入交辦事項，並內建核取方塊 (Checkbox) 供代課老師追蹤進度。

備忘錄空間： 為代課老師預留額外的空白區域，用於記錄課堂備註或突發狀況。

PDF/列印輸出： 透過瀏覽器的「列印」功能，可將排版優化後的內容一鍵輸出為乾淨的 PDF 文件，或直接列印成紙本。

簽名欄位： 文件底部自動包含教師簽名欄位，方便正式文件使用。

🚀 如何使用

1. 設置您的課表 (核心步驟)

您必須先編輯 HTML 檔案中的 JavaScript 變數，替換成您自己的課表資料。

使用文字編輯器（如 VS Code, Sublime Text 等）打開 checklist_generator.html 檔案。

找到 <script> 標籤中的 const weeklySchedule = { ... }; 區塊。

根據您的課表，修改或新增：

鍵（Key）：1 代表星期一，2 代表星期二，依此類推。

值（Value）：包含該日所有課程物件的陣列。

範例結構：

const weeklySchedule = {
    // 星期一
    1: [
        { period: "第一節", time: "08:10-08:50", subject: "國文", class: "三年一班" },
        // ... 其他課程
    ],
    // 星期二
    2: [ /* ... */ ],
    // ...
};


2. 運行與生成文件

在任何現代瀏覽器中（Chrome, Edge, Firefox 等）開啟 checklist_generator.html 檔案。

在頁面頂部選擇您要出差的日期。

點擊「1. 帶入當日課表」按鈕。

在生成的課表區塊中，直接點擊「交辦事項」和「備忘錄」區塊，輸入或編輯內容。

3. 輸出 PDF 或 Word

輸出 PDF (推薦)： 點擊「2. 產生PDF/列印」按鈕，在瀏覽器彈出的列印視窗中，將目的地設定為「另存為 PDF」，即可獲得排版整潔的紙本或電子檔。

輸出 Word (替代方案)： 手動選取頁面中生成的清單內容（從日期標題到簽名欄位），複製 (Ctrl+C) 後，貼上 (Ctrl+V) 到 Word 文件中。
