# AI 學術英文寫作 — 互動學習教材

> From Prompts to Paragraphs — 學術寫作五大核心原則

根據「AI 學術英文寫作公開課」逐字稿整理而成的互動式 Web App 學習教材。

## 功能

- **分段閱讀** — 5 個章節，含重點標註、範例對比、個人筆記
- **概念卡片** — 20 張可翻面的 Flashcard，支援已掌握/需複習標記
- **測驗** — 15 題填空與選擇題，即時回饋解析
- **寫作練習** — 4 個實戰練習（Paraphrasing、Hedging、Clarity、Literature Review）
- **進度追蹤** — 自動計算學習完成度

## 五大核心原則

1. **Evidence-Based** — 基於證據，不靠個人觀點
2. **Register** — 語域正式度，讓每個字都有意義
3. **Cautiousness** — 謹慎表達，善用 Hedging Devices
4. **Clarity** — 清晰度，Unclear writing implies unclear thinking
5. **AI 輔助** — AI 是 Coach 不是 Crutch

## 使用方式

這是一個純前端的單頁式 Web App（Single HTML File），不需要任何伺服器或安裝步驟。

### 線上瀏覽

部署至 GitHub Pages 後，直接開啟網址即可使用。

### 本機使用

直接用瀏覽器開啟 `index.html` 即可。

## 技術

- 純 HTML / CSS / JavaScript，無任何框架依賴
- 響應式設計，支援手機與桌面瀏覽
- 筆記功能使用 localStorage 本地儲存
