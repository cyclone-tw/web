# 分數著色互動學習單 - 版本紀錄

## 對話摘要

**日期**: 2025-12-11
**專案**: 分數圖形著色學習單 → 互動版升級

---

## 版本歷史

### v2.5.2 - 修正列印雙欄配置
- 增加 CSS 選擇器優先級
- 確保覆蓋 columns-2/3/4 class

### v2.5.1 - 列印版面優化
- 列印時強制雙欄配置
- 縮小題目卡片和圖形尺寸
- 一張 A4 可容納 6 題

### v2.5.0 - 列印模式新增題型選擇
- 新增「列印題型」下拉選單
- 支援三種列印題型：
  1. 看分數塗圖（空白圖形 + 顯示分數）
  2. 看圖寫分數（著色圖形 + 空白作答框）
  3. 混合出題

### v2.4.0 - 送出按鈕移至題目下方 + 列印成績單
- 送出答案按鈕移到題目區域下方
- 大型送出按鈕設計
- 結果頁新增「列印成績單」按鈕

### v2.3.0 - 新增學生名單下拉選單
- 建立 students.json 存放學生名單
- 學生姓名改為下拉選單
- 支援班級分組顯示

### v2.2.0 - 錯誤回饋增強
- 答錯時顯示正確答案（青色虛線框）
- 視覺化回饋標記（綠/紅/青色）

### v2.1.0 - 假分數修正 + 配色更新
- 假分數生成多個圖形
- 螢光綠/黃按鈕配色

### v2.0.0 - 互動版建立
- 三種模式：列印/看圖寫分數/看分數塗圖
- 真假分數選項
- 難度等級設定
- 計時功能
- Google Sheets 整合
- 暗色系主題

### v1.0.0 - 原始版本
- 分數著色.html（列印用）
