## 📖 使用教學

### 步驟 1：上傳圖片

```
1. 點擊左側「選擇圖片」按鈕
2. 選擇你的梗圖底圖（JPG、PNG、GIF）
3. 圖片會自動載入到畫布中
```

**提示**：建議使用 800x600 以內的圖片，載入速度更快！

---

### 步驟 2：新增文字框

```
1. 點擊「➕ 新增文字框」
2. 文字框會出現在畫布中央
3. 點擊文字框開始編輯
```

**可以新增無限個文字框！**

---

### 步驟 3：編輯文字內容

選中文字框後，左側會出現「文字樣式」設定區：

#### 📝 輸入文字
- 在「文字內容」欄位輸入你的梗圖文字
- 支援多行文字（按 Enter 換行）

#### 🎨 調整字體
```
黑體 - 預設，清晰易讀
明體 - 傳統優雅
Impact - 經典梗圖專用字體（推薦！）
Arial Black - 粗黑體，醒目
Comic Sans MS - 活潑可愛風格
等寬字體 - 程式碼或數字適用
```

#### 📏 調整大小
- 拖動「字體大小」滑桿：20-120px
- 或直接**拖曳文字框右下角**調整大小，文字會自動縮放！

#### 🎨 設定顏色
- **文字顏色**：點擊色塊選擇顏色（預設：白色）
- **描邊顏色**：設定外框顏色（預設：黑色）
- **描邊粗細**：0-10px（建議 3px 最經典）

#### 💪 加粗選項
勾選「粗體」讓文字更醒目

---

### 步驟 4：調整文字框位置

#### 🖱️ 拖曳移動
- 點擊文字框任意位置
- 按住滑鼠左鍵拖曳
- 放開滑鼠完成移動

#### 📐 縮放大小
- 點選文字框（出現藍色邊框）
- 拖曳**右下角的藍色圓點**
- 文字框和文字會**同步縮放**

#### 🗑️ 刪除文字框
- 選中文字框
- 點擊右上角的 **×** 按鈕

---

### 步驟 5：匯出梗圖

```
1. 完成編輯後，點擊「下載梗圖」
2. 梗圖會自動下載為 PNG 格式
3. 檔名格式：meme-時間戳.png
```

**所有文字效果（顏色、描邊、粗體）都會完整保留！**

```

```
想要多個相同樣式的文字框？
1. 設定好第一個文字框的樣式

## ❓ 常見問題

### Q1：為什麼拖曳文字框會跑出畫布外？
**A**：已設定邊界限制，文字框會自動停在畫布邊緣內。

### Q2：可以調整圖片大小嗎？
**A**：圖片會自動等比例縮放至 800x600 以內，確保最佳顯示效果。

### Q3：描邊效果在某些字體不明顯？
**A**：建議將描邊粗細調整到 4-6px，效果會更明顯。

### Q4：可以儲存編輯進度嗎？
**A**：目前版本暫不支援儲存進度，建議一次完成編輯並下載。

### Q5：匯出的圖片品質如何？
**A**：匯出為高品質 PNG 格式，完整保留所有視覺效果。

### Q6：支援 GIF 動圖嗎？
**A**：可以上傳 GIF，但匯出時會轉為靜態 PNG（取第一幀）。

### Q7：手機上可以使用嗎？
**A**：可以！介面支援觸控操作，但建議使用電腦以獲得最佳體驗。

### Q8：文字框可以旋轉嗎？
**A**：目前版本暫不支援旋轉，計畫在未來版本加入。
